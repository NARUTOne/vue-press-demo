# vuepress-demo

[vue-press](https://vuepress.vuejs.org/zh/)

## 开始

```sh

# 将 VuePress 作为一个本地依赖安装
yarn add -D vuepress # 或者：npm install -D vuepress

# 新建一个 docs 文件夹
mkdir docs

# 新建一个 markdown 文件
echo "# Hello VuePress!" > docs/README.md

# 开始写作
npx vuepress dev docs
# 或
npm run docs:dev

# build
npm run docs:build

```
